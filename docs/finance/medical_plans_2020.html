There are different rules governing how much you can contribute to an HSA vs. an FSA. In short, each plan holder is allowed to contribute $2700 in 2020 to an FSA (so that if you're on a family plan, you're stuck at $2700). On the flip side, if you're on a single-coverage HDHP, you can contribute $3550 in 2020 to an HSA, and $7100 if you have family coverage. Finally, note that contributions to an FSA are usually lost (although some plans allow for small roll-overs), so that you shouldn't contribute more to an FSA than your yearly expenses. On the other hand, HSAs roll over (and can accrue interest!), so there's no need to limit contributions based on yearly expenses. Finally, note that if one spouse contributes to an FSA, the other is no longer eligible to contribute to an HSA, even if the latter is covered by a HDHP. See this calculator's help page for more details.

Note that copays are counted differently than normal medical expenses. In general, copays count toward the out-of-pocket maximum, but not toward a deductible. See this calculator's help page for more details.
