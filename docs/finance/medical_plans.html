There are different rules governing how much you can contribute to an HSA vs. an FSA. In short, each plan holder is allowed to contribute $2650 in 2019 to an FSA (so that if you're on a family plan, you're stuck at $2650). On the flip side, if you're on a single-coverage HDHP, you can contribute $3500 in 2019 to an HSA, and $7000 if you have family coverage. Finally, note that contributions to an FSA are usually lost (although some plans allow for small roll-overs), so that you shouldn't contribute more to an FSA than your yearly expenses. On the other hand, HSAs roll over (and can accrue interest!), so there's no need to limit contributions based on yearly expenses. See this calculator's help page for more details.
